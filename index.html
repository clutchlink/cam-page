A better way to control cameras and DSLRs

Cam is a framework and Android app that can remotely control your camera.

News

August 2023: Rewrite, rename

In the spirit of companies rebranding to shorter names, CamControl will now be called Cam. The entire app is also being rewritten and will be published shortly. Improvements include:

  • Redesigned UI - much more responsive
  • Advanced File manager
  • Lua scripts and plugins - run on demand, or automatically
  • Use much less RAM and CPU resources

Camera as a Service (CaaS)

Remotely control a camera from anywhere in the world with a rich API over the web. Available for astronomy and long-term timelapse.

App Features

  • Support for modern Canon EOS cameras
  • Canon Liveview + custom Magic Lantern liveview
  • Get & Set ISO, aperture, shutter, speed, format, white balance
  • Take pictures and perform bulb shots
  • Download captured images
  • Manually and precisely focus lens
  • Basic File manager
  • Profiles - preconfigured settings & parameters for each camera / workflow
  • Scripting API with Javascript

Planned Features:

  • RAM Capture (capture, preview, and discard)
  • Support for Sony cameras
  • Use phone as external microphone
  • Clock sync
  • GPS Sync
  • Customizable Zebras/histogram overlays
